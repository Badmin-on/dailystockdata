# ✅ 수동 데이터 수집 가이드

## 🎉 Cron Jobs 문제 해결 완료!

**이제 Cron Jobs 없이 수동으로 데이터를 수집할 수 있습니다!**

---

## 🚀 사용 방법 (3단계)

### 1단계: Vercel 배포

**지금 바로 배포하세요:**

1. **Vercel Dashboard 접속**
   ```
   https://vercel.com/dashboard
   ```

2. **프로젝트 선택**: `dailystockdata`

3. **Deployments → Redeploy** 클릭
   - 또는 자동 배포 대기 (GitHub 푸시 후 자동 시작)

4. **배포 완료 대기** (약 3-5분)

---

### 2단계: 데이터 수집 페이지 접속

**배포 완료 후:**

```
https://your-project.vercel.app/admin/collect
```

또는 메인 페이지에서 **"🔧 데이터 수집"** 버튼 클릭

---

### 3단계: 매일 아침 데이터 수집

#### 📊 재무 데이터 수집 (먼저!)

1. **"🚀 재무 데이터 수집 시작"** 버튼 클릭
2. **진행 상황 실시간 확인**:
   - 진행률: 0/1000 → 1000/1000
   - 프로그레스 바
   - 실시간 로그
3. **완료까지 약 20-30분 대기**
4. ✅ **"모든 작업 완료!" 메시지 확인**

#### 💰 주가 데이터 수집 (그 다음!)

1. **"🚀 주가 데이터 수집 시작"** 버튼 클릭
2. **진행 상황 실시간 확인**:
   - 진행률: 0/1000 → 1000/1000
   - 프로그레스 바
   - 실시간 로그
3. **완료까지 약 5-7분 대기**
4. ✅ **"모든 작업 완료!" 메시지 확인**

## ✨ 장점

### 🎯 완전 제어

- ✅ **원하는 시간에 실행**: 매일 아침, 점심, 저녁 언제든
- ✅ **즉시 확인 가능**: 실시간 진행 상황
- ✅ **오류 즉시 대응**: 로그 확인 후 재시도

### 📊 실시간 모니터링

- ✅ **진행률 프로그레스 바**: 얼마나 진행됐는지 한눈에
- ✅ **실시간 로그**: 어떤 기업 수집 중인지 실시간 확인
- ✅ **통계 표시**: 수집 완료, 오류, 소요 시간

### 💰 비용 절감

- ✅ **Vercel Free Plan 가능**: Cron Jobs 제한 없음
- ✅ **Pro Plan 불필요**: $20/월 절약

### 🔧 유연성

- ✅ **선택적 수집**: 재무만 또는 주가만 수집 가능
- ✅ **재시도 간편**: 오류 시 즉시 다시 시도
- ✅ **브라우저만 있으면 OK**: 어디서든 접속 가능

## 🚨 주의 사항

### ⚠️ 수집 중 유의사항

1. **브라우저 창 닫지 마세요**
   - 수집이 중단됩니다
   - 완료까지 기다려주세요

2. **순서 지키세요**
   - 재무 데이터 먼저 → 주가 데이터 나중에
   - 동시에 실행하지 마세요

3. **완료 확인하세요**
   - "✅ 모든 작업 완료!" 메시지까지 확인
   - 중간에 멈추지 않도록 주의

4. **오류 발생 시**
   - 로그 확인
   - 잠시 후 다시 시도
   - 일부 기업 오류는 정상 (스크래핑 실패 가능)

## 🔄 데이터 갱신 확인

**수집 완료 후:**

1. **메인 페이지** (`/`) 접속
   - 등록 기업: 증가 확인
   - 재무 데이터: 증가 확인
   - 주가 데이터: 증가 확인

2. **투자 기회 발굴** (`/opportunities`) 접속
   - 최신 데이터로 업데이트된 기업 목록 확인
   - 점수 재계산 완료

3. **모니터링 대시보드** (`/monitor`) 접속
   - Top 20 기업 확인
   - 최신 데이터 반영 확인
